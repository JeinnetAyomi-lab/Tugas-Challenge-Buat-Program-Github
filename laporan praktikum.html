Slide 1 - 3

Pendahuluan & Konsep Dasar

OOP fokus pada Objek sebagai unit utama. Objek memiliki:

  • State: Data/Atribut (contoh: warna, model).
  • Behavior: Perilaku/Metode (contoh: berjalan, berhenti).

Perbedaan: Imperatif fokus pada langkah-langkah, sedangkan OOP fokus pada interaksi antar objek.

Slide 5 - 7

Sintaks Dasar Kelas & Objek

Membuat cetakan (Class) dan mewujudkannya menjadi instance (Object).

class Car {
  String? model;
  String? color;

  void start() => print('Car $model started!');
}

void main() {
  var myCar = Car(); 
  myCar.model = 'Sedan';
  myCar.start();
}
Slide 8 - 11

Constructor & 'this' Keyword

Constructor digunakan untuk inisialisasi properti secara otomatis. Keyword this merujuk pada atribut di dalam kelas itu sendiri.

class Mahasiswa {
  String nama;
  String nim;

  // Parameterized Constructor
  Mahasiswa(this.nama, this.nim);

  void tampil() => print('Nama: $nama, NIM: $nim');
}

void main() {
  var mhs = Mahasiswa('Jeinnet', '2026001');
  mhs.tampil();
}

Hasil Quiz & Diskusi (Slide 14)

1. Perbedaan Kelas dan Objek?
Kelas adalah blueprint/rancangan, objek adalah wujud nyata dari rancangan tersebut.

2. 3 Keuntungan OOP?
Modular (teratur), Reusable (bisa dipakai ulang), dan Maintainable (mudah dirawat).

3. Implementasi Kelas Hewan:

class Hewan {
  String nama;
  Hewan(this.nama);
  void bersuara() => print('$nama mengeluarkan suara!');
}
