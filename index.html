Ringkasan Materi

OOP adalah metode pemrograman yang berorientasi pada objek. Bayangkan Kelas (Class) sebagai cetakan kue, dan Objek (Object) sebagai kue yang sudah matang. Kelas menentukan bentuk dan rasa, sedangkan objek adalah hasil nyata yang bisa kita makan.

  • Attribute: Data di dalam kelas (contoh: nama, nim).
  • Method: Fungsi atau aksi (contoh: tampilkanData()).
  • Constructor: Cara instan untuk mengisi data saat objek pertama kali dibuat.

Contoh Implementasi Kode

Berikut adalah kode Dart yang menerapkan konsep Class, Constructor, dan Method untuk data Mahasiswa:

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  // Constructor menggunakan 'this' agar lebih ringkas
  Mahasiswa(this.nama, this.nim, this.ipk);

  void tampilkanData() {
    print('--- Data Mahasiswa ---');
    print('Nama : $nama');
    print('NIM  : $nim');
    print('IPK  : $ipk');
  }
}

void main() {
  var mhs1 = Mahasiswa("Jeinnet", "2026001", 3.95);
  mhs1.tampilkanData();
}

Hasil Quiz (Slide 14)

1. Apa perbedaan utama antara kelas dan objek?
Kelas adalah cetakan/rancangan (blueprint), sedangkan objek adalah hasil nyata (instance) dari cetakan tersebut yang sudah memiliki nilai data.
2. Sebutkan 3 keuntungan menggunakan OOP.
a. Reusable: Kode bisa digunakan berulang kali.
b. Organized: Kode lebih rapi karena terbagi per bagian (modul).
c. Maintainable: Lebih mudah diperbaiki jika terjadi error tanpa merusak bagian lain.
3. Buat kelas Hewan dengan atribut nama dan metode bersuara().
class Hewan {
  String nama;
  Hewan(this.nama);
  void bersuara() => print('$nama bersuara!');
}
